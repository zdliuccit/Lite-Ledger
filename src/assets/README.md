# 资源目录说明

- 日期：2026-04-26
- 执行者：Codex

## 目录约定

- `app/`：应用运行时实际使用的图片与图标资源。
- `ledger/`：账本模块运行时使用的图片资源。
- `sample/`：历史示例资源与 Expo 初始化资源，当前不作为应用正式资源引用。

## Ledger 资源说明

- `ledger/icons/tab/`：账本底部导航图标。
- `ledger/icons/feature/`：账本页面功能图标、列表图标和按钮图标。

## App Icon 说明

- `app/icon/icon.png`：通用应用图标，供 Expo 通用配置与 iOS 配置使用。
- `app/icon/icon-transparent.png`：Android 自适应图标前景图。
- `app/icon/favicon.png`：Web favicon。
- `app/icon/ios/AppIcon.appiconset/`：iOS 图标源目录。

## 同步说明

- 当前 iOS 原生目录 `ios/liteledger/Images.xcassets/AppIcon.appiconset/` 已同步为同一套图标资源。
