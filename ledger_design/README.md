# 轻账本 App 三倍高清设计资源包

## 资源说明
- screens_3x_png：每个页面单独导出，尺寸 1125 × 2436 px，对应 375pt @3x。
- icons_png_3x：独立 PNG 图标，192 × 192 px。
- icons_svg：独立 SVG 图标，适合开发直接引用或二次编辑。
- design_system：颜色、字体、间距、圆角、组件规范。
- components_3x_png：按钮、卡片等组件示例。
